GroundForge

Create prickings for generated diagrams.

The working diagrams generated by the recursive page don't care about positions of pins. The thumbnail page may provide one or more options for pin positions but there are infinite possibilities and you might not want a pin for each stitch. Download the shown patterns to edit. Nudge pins to the positions you prefer or otherwise customize a patch to your own taste and fit it into a pricking. Most important clue about reshaping patterns from the user guide: dots with the same color should match when "sewing" repeats together, they should also move together.
